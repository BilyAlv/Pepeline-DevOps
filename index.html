🛠️ Levantar el entorno local con Docker

Sigue estos pasos para configurar tu entorno de desarrollo local:

  1. Clona el repositorio:

    git clone https://github.com/Carocris/PipelineDevOps.git

  2. Crea un archivo .env en la raíz del proyecto:

    DB_USER=sa
DB_PASSWORD=ccoa1234!
DB_SERVER=db
DB_NAME=DevOpsDB

  3. Levanta los servicios:

    docker-compose up --build

  4. Accede al backend: http://localhost:3000
